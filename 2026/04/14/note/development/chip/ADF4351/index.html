2 ADF4351

ADF4351关键参数如下：

  • 输出频率范围：35MHz to 4400MHz
  • 内置压控振荡器（Voltage Controlled Oscillator，VCO），VCO的输出频率为2200MHz ~ 4400MHz。
  • 低相位噪声的VCO
  • 可编程增益输出
  • 参考输入频率范围为：10MHz to 250MHz，输入峰峰值范围为：0.7V to

2.1 ADF4351原理框图

ADF4351原理框图如下，由参考输入、PFD两个部分构成。

fref为，即外部参考时钟输入。在进入PFD之前，可配置2倍频、2分频和R分频器，这三个都是可选的。

PFD的两个输入应当设计为同一频率。鉴相器的频率范围如下：

  • 32MHz，分数N模式
  • 45MHz，整数N模式，且VCO频段选择使能（band select enabled）

VCO的输出到RFout，或反馈到N分频器前，可选一个DIVIDER，该分频器为可选系数如下：1、2、4、8、16、32、64。

2.3 VCO

根据手册可以知道，内部VCO工作频率范围是2.2 to 4.4GHz。

在计算的时候，已知和，需要计算一个合理的分频值，使得VCO工作在合理的范围内。

VCO核心由3个独立的VCO组成，每个VCO使用16个重叠的频段。如手册Figure 20所示。

to allow a wide frequency range to be covered without a large VCO sensitivity (KV) and resultant poor phase noise and spur ious performance.

翻译为：“以便在不需要较大 VCO 灵敏度 (KV) 的情况下覆盖宽频率范围，从而避免由此导致的相位噪声和杂散性能差的问题。”

2.3.1 频段选择band select

在上电或更新寄存器0时，VCO和band select logic会自动选择正确的VCO和频段。

VCO和频段选择需要花费10个PFD周期乘以频段选择时钟分频器的值。

R计数器的输出被用作频段选择逻辑的时钟，在R计数器的输出端提供了1 to 255的可编程分频器。

频段选择的时钟不宜太高，这会导致没有足够的时间进行正确的频段选择。

频段选择的时钟也不宜过低，这会导致锁定太慢。

该部分请详读手册。

2.4 双模分频器

ADF4351内置了两个双模分频器，4/5和8/9分频器。在配置时，只需要关心其限制，不同的预分频器模式，对 INT 值的下限有不同的要求，这源于内部的硬件限制。

  • 4/5预分频器，INT必须≥23
  • 8/9预分频器，INT必须≥75

当最大RF frequency达到3.6GHz时，则必须设置为8/9。

Deepseek对其的解释如下：

4/5或8/9代表的是分频比，芯片会将VCO输出的高频信号频率除以4或除以5（对于8/9预分频器则是除以8或除以9）

双模的含义：它不是一个固定的分频器，而是芯片会根据情况，自动在两个分频比之间切换

不需要做额外的计算，只需要选择一个模式，然后在INT中设置最终想要的分频因子N即可，芯片内部会完成剩下的工作。

2.6 鉴相器和电荷泵

鉴相器从R计数器和N计数器获取输入，并产生相位差比例的输出。如图所示

PFD包含一个可编程的延迟元件，用于设置防间隙脉冲的宽度。该脉冲用于确保PFD传递函数中没有死区。寄存器3（R3）中的DB22位按如下方式设置APB

  • DB22设置为0，ABP width 被编程为6ns，分数N应用的推荐值
  • DB22设置为1，ABP width 被编程为3ns，整数N应用的推荐值

对于整数N的应用，较短的脉冲宽度可以改善带内相位噪声。PFD频率可以运行到90MHz。

关于电荷泵，Deepseek解释如下： > 简单来说，电荷泵在ADF4351这类PLL芯片中，扮演着一个数模转换器（DAC）和执行器的角色。它将鉴相器（PFD）输出的“超前”或“滞后”的数字脉冲，转换成连续的模拟电流，去驱动外部的环路滤波器，从而控制VCO的频率。

4 寄存器配置

4.1 Reg0

16-Bit Integer Value (INT)

N分频器中的整数INT。INT最小值为23，在8/9预分频器下，最小值为75。

12-Bit Fractional Value (FRAC)

N分频器中的Frac。

4.2 Reg1

Phase Adjust

PH1 PHASE ADJ
0 OFF
1 ON

用于启用对特定输出频率的输出相位进行调整。 启用时，会跳过VCO频段选择（band select）和相位再同步（phase resync）两个步骤。 建议仅在固定频率或与原选择频率偏差小于1MHz的情况下禁用VCO频段选择。

Prescaler Value

PR1 PRESCALER
0 4/5
1 8/9

当预分频器被配置为4/5时，允许最大射频频率为3.6GHz。因此，当ADF4351在3.6GHz以上工作时，必须设置为8/9。 且预分频器对INT的值有如下限制：

  • Prescaler = 4/5：
  • Prescaler = 8/9：

12-Bit Phase Value

相位字必须小于REG1中的MOD值。相位字用于编程RF输出相位0° ~ 360°，分辨率为360°/MOD。在大多数情况下，RF信号与参考信号之间的相位关系并不重要。在这种应用中，相位值可以用于优化分数和子分数杂散水平。更多信息查阅手册的杂散一致性和分数杂散优化部分。如果既不使用相位重同步功能，也不使用杂散优化功能，建议将相位字设置为1。

12-Bit Modulus Value (MOD)

N计数器中的MOD。参考2.2 ADF4351的N分频器。

4.3 Reg2

Low Noise and Low Spur Modes

L2 L1 NOISE MODE
0 0 LOW NOISE MODE（低噪声模式）
0 1 RESERVED（保留）
1 0 RESERVED （保留）
1 1 LOW SPUR MODE（低杂散模式）

选择杂散模式时，抖动（dither）被启用。抖动将分数量化噪声随机，使其更像白噪声而非杂散噪声。低杂散模式用于PLL闭环带宽较宽的快速锁定应用。宽环路带宽是指大于通道不仅分辨率十分之一的环路带宽。为了获得最佳噪声性能，使用低噪声模式，抖动将被禁用。该模式确保电荷泵在噪声性能最佳区域内工作。低噪声模式可用于窄环路带宽滤波器。

MUXOUT

M3 M2 M1 OUTPUT
0 0 0 THREE-STATE OUTPUT，三态输出
0 0 1 DVDD
0 1 0 DGND
0 1 1 R COUNTER OUTPUT
1 0 0 N DIVIDER OUTPUT
1 0 1 ANALOG LOCK DETECT，模拟锁定信号输出
1 1 0 DIGITAL LOCK DETECT，数字锁定信号输出
1 1 1 RESERVED，保留

Reference Doubler

RD2 REFERENCE DOUBLER
0 DISABLED
1 ENABLED

如果关注杂散性能和相位噪声，请详读手册该部分。当启用该倍频器时，最大REFIN频率为30MHz。

RDIV2

在 R 计数器和PFD之间插入一个2分频器，可以扩展REFIN输入速率。此功能允许在PFD输入端出现50%占空比的信号，这是减少周期跳变所必须的。

10-Bit R Counter

参考2.1节，ADF4351原理框图

Double Buffer

双缓冲功能，参考手册“程序模式”部分。

Charge Pump Current Setting

电荷泵电流设置。

Lock Detect Function (LDF)

锁定检测功能，LDF控制锁定检测电路监控PFD周期数，以确定是否完成锁定。当该为设置为0时，监控PFD周期数为40。当该位设置为1时，监控PFD周期数为5。建议在分数-N模式下将该位设置为0，在整数模式下设置为1。

Lock Detect Precision (LDP)

锁定检测精度设置锁定检测电路中的比较窗口。该位设置为0时，比较窗口为10ns；设置为1时，比较窗口为6ns。当连续n个PFD周期小于比较窗口时，锁定检测电路输出高电平。对于整数-N模式，推荐设置为1。

Phase Detector Polarity

当使用无源环路滤波器或非反相有源环路滤波器时，该位设置为1。反之，设置为0。

Power-Down (PD)

将此位设置为1立即执行掉电。在控制该位掉电时，寄存器的值会保留，而不会丢失。

Charge Pump Three-State

通常该位被设置为0。

Counter Reset

该位置1时，R计数器和N计数器被复位。通常该位被设置为0。

4.4 Reg3

Band Select Clock Mode

频段选择时钟模式，设置为1会选择更快的频段选择逻辑序列，这适用于PFD频率较高时，并且对于快速锁定是必要的。对于PFD小于125kHz的情况，该位设置为0。也就是说，频段选择逻辑的时钟，最好是不大于125kHz。

Antibacklash Pulse Width (ABP)

抗反脉冲宽度，用于确保PFD传递函数中不存在死区。该位设置为 1 时，PFD 抗反脉冲宽度为 3 ns，这将在整数 N 应用中改善相位噪声和杂散。对于小数-N 模式，不推荐使用 3 ns 的设置。

Charge Cancelation

电荷抵消，在整数 N 模式下，减少PFD毛刺。在分数 N 模式下，应将此位设置为 0。

CSR Enable

Cycle Slip Reduction 周期跳变抑制。将该位设置为 1 可启用周期跳变抑制。CSR 是一种用于提高快速锁定时间的方法。PFD 的信号必须具有50%的占空比，CSR才能够生效。电荷泵电流也必须设置为最小值。更多信息请阅读手册。

Clock Divider Mode

C2 C1 CLOCK DIVIDER MODE
0 0 CLOCK DIVIDER OFF
0 1 FAST LOCK ENABLE
1 0 RESYNC ENABLE
1 1 RESERVED

12-Bit Clock Divider Value

参考快速锁定、相位重同步部分，通常设置为0即可。

4.5 Reg4

Feedback Select

该位用于选择从 VCO 输出到 N 计数器的反馈。设置为 1 时，信号直接取自 VCO，设置为 0 时，信号取自分频器输出，此时两个信号输出是同相的，在某些信号正干涉以增加功率的应用中，这非常有用。

RF Divider Select

D12 D11 D10 RF DIVIDER SELECT
0 0 0 ÷1
0 0 1 ÷2
0 1 0 ÷4
0 1 1 ÷8
1 0 0 ÷16
1 0 1 ÷32
1 1 0 ÷64

Band Select Clock Divider Value

该值为频段选择逻辑的时钟输入设置分频器。默认情况下，R 计数器的输出用于频段选择器时钟，但是如果高于 125kHz，可以开启分频器，将 R 计数器的输出除以一个较小的值。

VCO Power-Down

该位设置为 1 时关闭VCO。

Mute Till Lock Detect (MTLD)

该位设置为 1 时，锁定后才输出。

AUX Output Select

辅助射频输出，该位设置为 0，辅助射频输出是射频分频器的输出；若设置为 1，则是 VCO 直接输出。

AUX Output Enable

使能辅助射频输出将该位设置为1。

AUX Output Power

D5 D4 AUX OUTPUT POWER
0 0 –4dBm
0 1 –1dBm
1 0 +2dBm
1 1 +5dBm

RF Output Enable

开启主射频输出，将该位设置为1。

Output Power

D5 D4 RF OUTPUT POWER
0 0 –4dBm
0 1 –1dBm
1 0 +2dBm
1 1 +5dBm

4.6 Reg5

Lock Detect Pin Operation

D15 D14 LOCK DETECT PIN OPERATION
0 0 LOW，输出低电平
0 1 DIGITAL LOCK DETECT，数字锁定信号输出
1 0 LOW，输出低电平
1 1 HIGH，输出高电平
